TASK 2 - API Automation task

How to run a Test

1. Go to a root of test folder, in terminal console type: npm install
   ( wait while all dependencies will install )

2. In a terminal console type: npm test

3. The test will run


///////////////////////////////////////////// BONUS QUESTION /////////////////////////////////////////////

 
BONUS QUESTION: How will you resolve the issue with long-time test execution?

We can use timeout to give a test function to increase a time to complete an operation:
  
For example:

jest.setTimeout(30000)
